Class: Button

Button(button, action)

new Button(button, action)

Enable button functionality.
Parameters:
Name Type Description
button DOMElement The DIV element object you want the dropdowns to be in. It needs to have "dropdown" as the first parameter of the class attribute. Passed as reference.
action function What the button will do when the button is released (after pressing down). The MouseEvent object would be passed to this function.

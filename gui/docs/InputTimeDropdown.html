Class: InputTimeDropdown

InputTimeDropdown(container, type, timeObj)

new InputTimeDropdown(container, type, timeObj)

Generate graphics for the time selector. Make sure its parent element is a block element.
Parameters:
Name Type Description
container DOMElement The DIV element object you want the dropdowns to be in. It needs to have "dropdown" as the first parameter of the class attribute. Passed as reference.
type string Specify if the dropdown value is in hours (hr), minutes (min), or seconds (sec).
timeObj Time The time object you want to control. Passed as reference.

Class: Time

Time(timeStr)

new Time(timeStr)

Stores the time with hours, minutes, and seconds from 00:00:00 to 23:59:59. It is modifiable and can convert to select data formats. Make sure the time values do not exceed +/-( 2^32/2 - 1 ).
Parameters:
Name Type Description
timeStr string Time in hh:mm:ss or hh:mm.
